# PrimeNG Splitter Test

Minimal Angular + PrimeNG test project.

## Task

- Use a standard PrimeNG `Splitter` with three blocks.
- Place a standard PrimeNG `MultiSelect` checkbox dropdown in one block.
- Keep PrimeNG components in their native structure without custom overlays or internal style overrides.

## Local Development

```bash
npm install
npm start
```

Open `http://localhost:4200`.

## Build

```bash
npm run build
```

## GitHub Pages

Every push to `main` is built and published by `.github/workflows/deploy-pages.yml`.
